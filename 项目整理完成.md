
```bash

git add docs/
```

#### 第四步：验证
```bash
# 查看将要提交的文件
git status

```


## 🚀 发布流程

### 1. 测试构建系统
```bash
python test\test_multi_version.py
```

### 2. 构建所有版本
```bash
# 方法A: Python脚本
python scripts\build_multi_version.py --all

# 方法B: 批处理
scripts\打包所有版本.bat
```

### 3. 测试所有版本
- CPU版本测试
- GPU-CUDA11版本测试
- GPU-CUDA12版本测试

### 4. 压缩打包
```powershell
cd releases
7z a MEMEFinder_cpu_v1.0.0.zip MEMEFinder_cpu\*
7z a MEMEFinder_gpu-cuda11_v1.0.0.zip MEMEFinder_gpu-cuda11\*
7z a MEMEFinder_gpu-cuda12_v1.0.0.zip MEMEFinder_gpu-cuda12\*
```

### 5. 发布到GitHub Release

按照 [发布完整教程.md](发布完整教程.md) 第5章操作：

1. 创建Tag: `v1.0.0`
2. 创建Release
3. 上传文件（按顺序）：
   - MEMEFinder_cpu_v1.0.0.zip
   - MEMEFinder_gpu-cuda11_v1.0.0.zip
   - MEMEFinder_gpu-cuda12_v1.0.0.zip
4. 填写发布说明
5. 发布

详细步骤请参考完整教程。
