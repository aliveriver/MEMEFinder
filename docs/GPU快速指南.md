# ⚡ GPU加速 vs CPU模式 - 快速选择指南

## 🚀 我想使用GPU加速（速度快3倍）

运行GPU修复向导：
```bash
python gpu_fix_wizard.py
```

向导会：
- ✅ 检查你的GPU环境
- ✅ 找出问题所在
- ✅ 提供修复方案
- ✅ 可选自动修复

修复后，程序会自动使用GPU加速！

---

## 🛡️ 我只想程序能运行（稳定优先）

运行快速修复工具：
```bash
python fix_gpu_crash.py
```

选择 `[2] 使用CPU模式`，然后用生成的启动器运行程序。

---

## 🤔 我不确定，先测试一下

运行：
```bash
python fix_gpu_crash.py
```

选择 `[3] 只是测试一下`，看看你的GPU状态。
