## 📋 快速开始

### 1. 安装依赖
```powershell
.\安装依赖.bat
# 或
pip install -r requirements.txt
```

### 2. 启动程序
```powershell
.\启动程序.bat
# 或
python meme_finder_gui.py
```

### 3. 使用步骤
1. 点击"➕ 添加图源文件夹"
2. 选择包含表情包的文件夹
3. 点击"🔍 扫描新图片"
4. 查看统计信息

### 4. 测试功能
```powershell
python test_gui.py
```

## 💡 使用建议

### 推荐的图源文件夹
- 微信/QQ表情包保存目录
- 截图文件夹
- 下载的表情包合集
- 自己制作的表情包

### 性能建议
- 首次扫描建议分批添加图源
- 单个图源建议不超过5000张图片
- 定期清理不需要的图源

### 数据备份
```powershell
# 备份数据库
copy meme_finder.db meme_finder_backup.db
```
