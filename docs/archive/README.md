# Archive 目录说明

此目录包含项目开发过程中的历史文档，这些文档记录了项目的演进历史，但不再作为主要参考文档使用。

## 📚 文档分类

### Bug 修复记录
- **`BUG_FIXES.md`** - 历史 Bug 修复记录
- **`GPU_CRASH_FIX_SUMMARY.md`** - GPU 崩溃修复总结
- **`GPU_FINAL_FIX.md`** - GPU 问题最终修复方案
- **`GPU_FIX_GUIDE.md`** - GPU 修复指南
- **`SNOWNLP_FIX_SUMMARY.md`** - SnowNLP 修复总结
- **`SNOWNLP_PACKAGING.md`** - SnowNLP 打包文档

### 项目管理文档
- **`PROJECT_COMPLETION.md`** - 项目完成情况记录
- **`REFACTOR_SUMMARY.md`** - 重构总结
- **`RELEASE_TEMPLATE.md`** - 发布模板

### 优化文档
- **`OPTIMIZATION_CHECKLIST.md`** - 优化检查清单
- **`OPTIMIZATION_README.md`** - 优化说明文档

## 💡 当前有效文档

- **根目录**: `README.md` - 项目主文档
- **docs/**:

  - `QUICK_START.md` - 快速开始

这些归档文档保留下来是为了：
1. 记录项目演进历史
