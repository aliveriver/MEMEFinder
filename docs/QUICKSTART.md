# 🚀 MEMEFinder 快速入门 (5分钟)

## 第一步：安装依赖 (2分钟)

双击运行：
```
安装依赖.bat
```

或在PowerShell中运行：
```powershell
pip install -r requirements.txt
```

## 第二步：启动程序 (1分钟)

双击运行：
```
启动程序.bat
```

或在PowerShell中运行：
```powershell
python meme_finder_gui.py
```

你会看到这样的界面：

```
┌─────────────────────────────────────────┐
│    表情包查找器 - MEMEFinder              │
├─────────────────────────────────────────┤
│ 📁图源管理 │ ⚙️图片处理 │ 🔍图片搜索     │
├─────────────────────────────────────────┤
│                                         │
│  ➕ 添加图源  🗑️ 删除  🔄 刷新  🔍 扫描   │
│                                         │
│  ID │ 路径 │ 添加时间 │ 扫描时间 │ 状态  │
│  ─────────────────────────────────────  │
│  (空列表)                                │
│                                         │
│  统计: 总图片: 0 | 已处理: 0 | 未处理: 0 │
└─────────────────────────────────────────┘
```

## 第三步：添加图源 (1分钟)

1. 点击 **"➕ 添加图源文件夹"**
2. 选择你的表情包文件夹（例如：`D:\MEMEFinder\imgs`）
3. 点击"选择文件夹"

图源会立即显示在列表中！

## 第四步：扫描图片 (1分钟)

1. 点击 **"🔍 扫描新图片"**
2. 等待扫描完成（几秒钟）
3. 查看弹窗提示：`发现新图片: XX 张`

完成！你的图片已经录入系统。

## ❓ 遇到问题？

### 问题1: 程序打不开
**解决方法：**
```powershell
# 检查Python版本
python --version

# 应该显示 Python 3.8 或更高版本
```

### 问题2: 依赖安装失败
**解决方法：**
```powershell
# 升级pip
python -m pip install --upgrade pip

# 重新安装
pip install -r requirements.txt
```

### 问题3: 扫描没有发现图片
**可能原因：**
1. 文件夹里没有支持的图片格式
2. 图片已经扫描过了（有去重）

**支持的格式：**
- .jpg / .jpeg
- .png
- .bmp
- .webp
- .gif
- .tiff
